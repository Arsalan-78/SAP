EXECUTIVE SUMMARY

Intelligent Invoice Processing
for Modern SAP Environments

The SAP Invoice Agent is a production-ready, end-to-end automation solution that transforms manual Accounts Payable workflows into an intelligent, AI-driven process.

It seamlessly integrates n8n orchestration, FastAPI backend, LangGraph-powered decision agent (using Groq’s Llama 3.3 70B), and direct SAP OData connectivity to deliver accurate, explainable, and fully auditable invoice processing.
Core Value Delivered
  • Automatic 3-way matching with SAP Purchase Orders
  • AI-generated plain-English explanations for mismatches
  • Instant auto-posting to SAP for clean invoices
  • Smart escalation to human review when needed

CORE COMPONENTS

Built with Production-Grade Technologies

FastAPI Backend
app/main.py

High-performance API server that receives invoice uploads from n8n, orchestrates the full pipeline, and returns structured decisions.

Endpoints: /process-invoice
LangGraph AI Agent
app/agent.py

Stateful decision engine using LangGraph. Performs 3-way matching, auto-approves clean invoices, and generates human-readable explanations for mismatches.

match_po approve explain_mismatch
OCR & Field Extractor
app/ocr_extractor.py

Uses Tesseract + pdf2image to extract text from PDFs and regex-based parsing to pull invoice number, PO number, vendor, and total amount.

SAP Connector
app/sap_connector.py

Full OData integration with API_PURCHASEORDER_PROCESS_SRV and invoice posting. Includes powerful mock mode with realistic sample POs for instant demos.

MOCK_MODE = True by default
n8n Orchestration
n8n/invoice-workflow.json

Ready-to-import workflow that handles webhook triggers, calls the FastAPI service, routes based on AI decisions, and sends professional email notifications.

INTELLIGENT WORKFLOW

How the Agent Processes an Invoice

1
Invoice Received
n8n webhook receives PDF (or email trigger). File is forwarded to FastAPI.
2
OCR & Data Extraction
Tesseract OCR extracts text → Structured fields parsed (Invoice #, PO #, Vendor, Amount).
3
LangGraph Decision Engine
Agent fetches PO from SAP and performs intelligent 3-way matching. Routes to auto-approve, explain mismatch, or escalate.
4
SAP Action + Notification
Clean invoices are posted automatically. Mismatches receive AI-generated explanations and are routed for human review via email.
